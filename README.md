## Description
Intégration Home Assistant locale pour les modules MOES Mini Joli Intelligent (dimmer, switch, curtain, garage).

## Installation
- Via HACS : ajouter le dépôt `SoFarSoGood86/moes_mini_module` comme dépôt personnalisé.
- Manuel : copier `custom_components/moes_mini_module` dans `config/custom_components/` et redémarrer HA.
