MOES Mini Module (local) – Version v1.1.0
Auteur : SoFarSoGood86
-----------------------------------------
Installation :
1. Extraire le fichier moes_mini_module_component_only.zip
2. Copier le dossier "moes_mini_module" dans le répertoire "custom_components" de Home Assistant
3. Redémarrer Home Assistant
4. Ajouter l'intégration via l'interface ou via configuration.yaml

Dépôt GitHub :
https://github.com/SoFarSoGood86/moes_mini_module
